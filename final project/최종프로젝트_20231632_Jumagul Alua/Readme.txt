/*-----------한국어------------*/
실행 후 'Start'버튼 누른다

속도를 선택하기:
-빨간 색: 빠른 속도
-주황 색: 중간 속도
-노란 색:느린 속도

게임 플레이:
초록 색 뱀이 빨간 색인 목표물3에 이동하는 과정 지켜본다
목표물을 먹으면 점수가 올라간다
뱀이 자신의 몸이나 벽에 부딪히면 게임 오버가 된다

일시 정지 및 게임 종료:
일시 정지를 원하면 'P'키를 누르고
게임 나가려면, 'Esc'키를 누른다

게임 오버 후:
다시 그 속도로 재시작을 원하면 'R'키를 누르고
게임 나가려면, 'Q'키를 누른다.

/*------------english------------*/
Click the 'Start' button to begin the game

Selecting Speed:
-Red: Fast speed
-Orange: Medium speed
-Yellow: Slow speed

Gameplay:
Watch the colored snake move towards the red target
The score increases when the snake eats the target
The game will be over if the snake collides with itself or the wall

Pause and Exit:
Press 'P' to pause the game
Press 'Esc' to exit the game

After Game Over:
Press 'R' to restart the game at the same speed
Press 'Q' to quit the game
